# decoraliaEcommerce
Decoralia E-Commerce
Bu proje ev dekorasyonu ürünlerinin tanıtımı,satışı  için yapılmış bir e-ticaret sitesidir. Veritabanı olarak Mysql kullanılmıştır. Kullanıcılar çeşitli rollerle ve yetkinliklerle siteye giriş yapabilmektedir.
Requirements
PHP >= 5.6.4
MySQL >= 5.7
OpenSSL PHP Extension
PDO PHP Extension
Mbstring PHP Extension
Tokenizer PHP Extension
XML PHP Extension
Install
Clone repo

Install Composer

Download Composer

composer update/install

composer install
Install Nodejs

Download Node.js

NPM dependencies

npm install
Using Laravel Mix

npm run dev
How to setting
Go into .env file and change Database and Email credentials.

php artisan migrate
php artisan db:seed
Generating a New Application Key

php artisan key:generate
